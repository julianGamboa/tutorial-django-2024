Plano de Estudos - Desvendando o Mundo da Programação e Banco de Dados

Objetivo:

Dominar as ferramentas essenciais para construir aplicações web robustas e bancos de dados eficientes.

Roteiro de Aprendizagem:

1. Mergulhando no Universo do Banco de Dados:

SQLite:

  • Gerenciar estudos com simplicidade e praticidade.
  • Contabilizar dias de estudo com precisão e organização.

Consultas SQL:

  • Criar consultas personalizadas para buscar informações.
  • Exibir livros por autor, gênero e muito mais!

CRUD:

  • Dominar operações CRUD (Criar, Ler, Atualizar e Deletar).
  • Manipular tabela de "verbos HTTP" com total controle.

2. Django: Desvendando os Segredos do Desenvolvimento Web:

Tutorial Django:

  • Criar aplicações web dinâmicas e escaláveis.
  • Iniciar e estruturar projetos Django com maestria.

3. Leitura Complementar:

MVC:

Aprofundar conhecimentos sobre o padrão MVC (Modelo-Visão-Controlador).

MVT:

Explorar o padrão MVT (Modelo-Visão-Template) do Django.

Exemplo Django:

Observar como os conceitos aprendidos se aplicam em um cenário real.

4. Desbravando o Front-End com HTML:

Criar seis páginas HTML essenciais para gerenciar seus livros:

  • actualizar_libro.html
  • consultar_libros.html
  • crear_autor.html
  • crear_libro.html
  • eliminar_libro.html
  • listar_libros.html

Lembre-se:

  • Este plano de estudos é um guia flexível.
  • Utilize ferramentas online e tutoriais.
  • Pratique consistentemente.
  • Divirta-se durante a jornada de aprendizado!

Recursos Adicionais:
